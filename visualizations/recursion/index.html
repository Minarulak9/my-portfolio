Understanding Recursion

What is Recursion?

Recursion is when a function calls itself to solve a problem by breaking it down into smaller subproblems.

Key Concepts:

  • Base Case: The condition that stops recursion
  • Recursive Case: The function calling itself
  • Call Stack: Memory structure storing function calls
  • Stack Frame: Memory allocated for each function call
